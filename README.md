# CodeBuildShipKe

##### Codelab demo

## Steps

### 1. Apk splitting
Why?
Smaller is always better :)
```
splits {
        abi {
            enable gradle.startParameter.taskNames.contains(":app:assembleRelease") || gradle.startParameter.taskNames.contains("assembleRelease")
            reset()
            include 'x86', 'x86_64', 'armeabi-v7a', 'arm64-v8a'
            universalApk true
        }
    }
```
